# MSExcel-Portfolio

LEGO Dashboard – analiza danych i KPI w Excelu/ Data and KPI analysis in Excel

Opis projektu:
Interaktywny dashboard Excel pokazujący dane o zestawach LEGO: liczba elementów, ceny, zestawy licencjonowane i kluczowe KPI.
Zawiera: slicer, mini KPI z kolorami (Power Pivot + formatowanie warunkowe) i wykresy przestawne oraz zwykłe.
Plik Excel gotowy do otwarcia i interakcji.

Project description:

Interactive Excel dashboard showing LEGO set data: pieces, prices, licensed sets, and key KPIs.
Includes slicer, color-coded mini KPIs (Power Pivot + conditional formatting), both pivot and  regular charts.
Excel file ready to open and explore.
